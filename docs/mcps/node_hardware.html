Node Hardware

System Management

Comprehensive system monitoring and hardware information with remote node support and health checks. Provides detailed CPU, memory, and system information for HPC environments.

1.0.0 Version
11 Actions
Jul 19 Updated

Available Actions

  • get_cpu_info
    Get comprehensive CPU information including specifications, core configuration, frequency analysis, and performance metrics
    Parameters: None
  • get_memory_info
    Get comprehensive memory information including capacity, usage patterns, and performance characteristics
    Parameters: None
  • get_system_info
    Get comprehensive system information including OS details, kernel version, and platform specifications
    Parameters: None
  • get_disk_info
    Get comprehensive disk and storage information including capacity, usage, and performance metrics
    Parameters: None
  • get_network_info
    Get comprehensive network information including interfaces, configurations, and connectivity status
    Parameters: None
  • get_gpu_info
    Get comprehensive GPU information including specifications, memory, and performance metrics
    Parameters: None
  • get_sensor_info
    Get comprehensive sensor information including temperature, voltage, and fan speed monitoring
    Parameters: None
  • get_process_info
    Get comprehensive process information including resource usage and system performance
    Parameters: None
  • get_performance_info
    Get comprehensive performance information including load averages, uptime, and system metrics
    Parameters: None
  • get_remote_node_info
    Get hardware information from remote nodes via SSH with comprehensive monitoring capabilities
    Parameters: hostname (str), username (str, optional), ssh_key (str, optional), port (int, optional), timeout (int, optional), components (list, optional)
  • health_check
    Perform comprehensive system health check with customizable thresholds and alerting
    Parameters: cpu_threshold (float, optional), memory_threshold (float, optional), disk_threshold (float, optional), include_detailed_report (bool, optional)

Usage Examples

Basic Hardware Information

// Get CPU information get_cpu_info() // Get memory details get_memory_info() // Get system overview get_system_info() // Get storage information get_disk_info()

Advanced Hardware Monitoring

// Monitor network interfaces get_network_info() // Check GPU status get_gpu_info() // Monitor system sensors get_sensor_info() // Check running processes get_process_info() // Get performance metrics get_performance_info()

Remote Hardware Information

// Get remote node information get_remote_node_info(hostname: "server1.example.com") // Remote monitoring with SSH key get_remote_node_info( hostname: "192.168.1.100", username: "admin", ssh_key: "~/.ssh/id_rsa" ) // HPC cluster monitoring get_remote_node_info( hostname: "hpc-node-01", username: "hpcuser", ssh_key: "~/.ssh/hpc_key", timeout: 120, components: ["cpu", "memory", "gpu"] )

System Health Monitoring

// Basic health check health_check() // Health check with custom thresholds health_check( cpu_threshold: 80.0, memory_threshold: 85.0, disk_threshold: 90.0 ) // Detailed health assessment health_check( cpu_threshold: 75.0, memory_threshold: 80.0, disk_threshold: 85.0, include_detailed_report: true )
