Compression

Data Processing

File compression utility using gzip compression. Provides efficient file compression capabilities for scientific computing workflows.

1.0.0 Version
1 Actions
Jan 15 Updated
Python Language

Installation Methods

Global Installation

uv pip install scientific-mcps

Individual MCP Installation

uv pip install "scientific-mcps[compression]"

Direct Server

uv run mcp_compression

Claude Desktop Configuration

{ "mcpServers": { "compression": { "command": "uv", "args": ["run", "mcp_compression"] } } }

Available Actions

  • compress_file
    Compress a file using gzip compression
    Parameters: file_path (str)

Usage Examples

Basic File Compression

// Compress a data file compress_file(file_path: "/data/large_dataset.csv") // Compress log files compress_file(file_path: "/logs/application.log") // Compress simulation results compress_file(file_path: "/results/simulation_output.txt")

Integration with Other MCPs

// Export data from pandas save_data("/data/processed_results.csv", "analysis_results.csv") // Compress the exported file compress_file("analysis_results.csv") // Use in data processing pipelines filter_data("/data/raw_data.csv", {"temperature": "> 25"}) compress_file("/data/filtered_data.csv")
