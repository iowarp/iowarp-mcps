Adios

Data Processing

Advanced I/O system for scientific data reading from multiple file formats. Provides high-performance data access for large-scale scientific simulations and analyses.

Available Actions

  • list_bp5
    Lists all BP5 files in a given directory, the bp5 files are actually directories so both file and directory words are correct
    Parameters: directory (str, default="data/") - must be an absolute path
  • inspect_variables
    Inspects variables in a BP5 file. If variable_name is provided, returns data for that specific variable. Otherwise, shows type, shape, and steps for all variables
    Parameters: filename (str) - absolute path to BP5 file, variable_name (str, optional) - specific variable name
  • inspect_variables_at_step
    Inspects a specific variable at a given step in a BP5 file. Shows variable type, shape, min, max. All parameters are required
    Parameters: filename (str) - absolute path, variable_name (str) - variable name, step (int) - step number
  • inspect_attributes
    Reads global or variable-specific attributes from a BP5 file
    Parameters: filename (str) - absolute path, variable_name (str, optional) - variable name for targeted inspection
  • read_variable_at_step
    Reads a named variable at a specific step from a BP5 file. All parameters are required
    Parameters: filename (str) - absolute path, variable_name (str) - variable name, target_step (int) - time step number

Usage Examples

Reading Scientific Data

// List available BP5 files list_bp5(directory: "/data/simulations/") // Inspect variables in a simulation file inspect_variables( filename: "/data/simulation_output.bp", variable_name: "temperature" ) // Read variable at specific time step read_variable_at_step( filename: "/data/simulation_output.bp", variable_name: "temperature", target_step: 10 ) // Inspect variable properties at a step inspect_variables_at_step( filename: "/data/simulation_output.bp", variable_name: "pressure", step: 25 ) // Get file attributes and metadata inspect_attributes("/data/simulation_output.bp")

Integration with Analysis Tools

// Read ADIOS data at multiple time steps read_variable_at_step("/data/simulation.bp", "pressure_field", 0) read_variable_at_step("/data/simulation.bp", "pressure_field", 50) // Inspect variable structure across steps inspect_variables("/data/simulation.bp", "temperature") // Get comprehensive file information inspect_attributes("/data/simulation.bp") list_bp5("/data/simulation_runs/")
