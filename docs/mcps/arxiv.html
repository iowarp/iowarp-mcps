ArXiv

Data Processing

Fetch and search research papers from the ArXiv repository. Access scientific literature, abstracts, and metadata for research and analysis purposes.

Available Actions

  • search_arxiv
    Search ArXiv papers by query with advanced filtering and result customization
    Parameters: query (str), max_results (int, default=10), sort_by (str, optional), sort_order (str, optional)
  • get_recent_papers
    Get recent papers from specific ArXiv categories with date filtering
    Parameters: categories (list), days_back (int, default=7), max_results (int, default=20)
  • search_papers_by_author
    Search for papers by specific author name with flexible matching
    Parameters: author_name (str), max_results (int, default=20), exact_match (bool, default=False)
  • search_by_title
    Search papers by title keywords with advanced text matching
    Parameters: title_keywords (str), max_results (int, default=15), fuzzy_match (bool, default=True)
  • search_by_abstract
    Search papers by abstract content with semantic matching capabilities
    Parameters: abstract_keywords (str), max_results (int, default=15), include_summary (bool, default=True)
  • search_by_subject
    Search papers by ArXiv subject categories with hierarchical browsing
    Parameters: subject_categories (list), max_results (int, default=25), include_subcategories (bool, default=True)
  • search_date_range
    Search papers within specific date ranges with flexible date formatting
    Parameters: start_date (str), end_date (str), categories (list, optional), max_results (int, default=20)
  • get_paper_details
    Get comprehensive details for a specific paper including metadata and metrics
    Parameters: arxiv_id (str), include_references (bool, default=False)
  • export_to_bibtex
    Export paper citations to BibTeX format for reference management
    Parameters: arxiv_ids (list), output_file (str, optional)
  • find_similar_papers
    Find papers similar to a given paper using content analysis
    Parameters: reference_paper_id (str), max_results (int, default=10), similarity_threshold (float, default=0.7)
  • download_paper_pdf
    Download PDF of a specific paper with automatic file management
    Parameters: arxiv_id (str), download_path (str, optional)
  • get_pdf_url
    Get direct PDF URL for a paper without downloading
    Parameters: arxiv_id (str)
  • download_multiple_pdfs
    Download multiple papers in batch with progress tracking
    Parameters: arxiv_ids (list), download_directory (str), max_concurrent (int, default=3)

Usage Examples

Literature Search

// Search for papers on machine learning search_papers( query: "machine learning quantum computing", category: "quant-ph", max_results: 10, sort_by: "relevance" ) // Fetch specific paper details fetch_paper( arxiv_id: "2301.12345", include_abstract: true, include_authors: true ) // Get paper metadata get_metadata("2301.12345", ["title", "authors", "published", "categories"])

Research Workflow

// Search for recent papers search_papers("computational physics", max_results: 20) // Extract paper data for analysis papers_data = [] for paper in results: metadata = get_metadata(paper.id) papers_data.append(metadata) // Analyze with pandas load_csv(papers_data) describe(dataframe, ["publication_date", "citation_count"])
