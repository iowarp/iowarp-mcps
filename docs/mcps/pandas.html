Available Actions

  • load_data
    Load and parse data from multiple file formats (CSV, Excel, JSON, Parquet, HDF5) with intelligent parsing capabilities
    Parameters: file_path (str), file_format (str, optional), sheet_name (str, optional), encoding (str, optional), columns (list, optional), nrows (int, optional)
  • save_data
    Save data to various file formats with optimization and format-specific configurations
    Parameters: data_file (str), output_file (str), output_format (str, optional), sheet_name (str, optional), index (bool, optional)
  • statistical_summary
    Generate comprehensive statistical summaries with descriptive statistics, distribution analysis, and data profiling
    Parameters: file_path (str), columns (list, optional), include_distributions (bool, optional)
  • correlation_analysis
    Perform advanced correlation analysis with multiple methods and statistical significance testing
    Parameters: file_path (str), columns (list, optional), method (str, optional), significance_test (bool, optional)
  • hypothesis_testing
    Perform statistical hypothesis testing with multiple test types and comprehensive statistical analysis
    Parameters: file_path (str), test_type (str), columns (list), groups (list, optional), alpha (float, optional)
  • handle_missing_data
    Handle missing values through various strategies including imputation, deletion, and advanced statistical methods
    Parameters: file_path (str), strategy (str), columns (list, optional), output_file (str, optional)
  • clean_data
    Comprehensive data cleaning including duplicates removal, outlier handling, and data validation
    Parameters: file_path (str), remove_duplicates (bool, optional), handle_outliers (bool, optional), output_file (str, optional)
  • groupby_operations
    Perform advanced group-by operations with multiple aggregation functions and statistical analysis
    Parameters: file_path (str), group_columns (list), aggregations (dict), output_file (str, optional)
  • merge_datasets
    Merge multiple datasets using various join operations with intelligent key detection and data validation
    Parameters: left_file (str), right_file (str), join_keys (list), join_type (str, optional), output_file (str, optional)
  • pivot_table
    Create sophisticated pivot tables with multiple aggregation functions and advanced reshaping capabilities
    Parameters: file_path (str), index_columns (list), value_columns (list), agg_functions (list, optional), output_file (str, optional)
  • time_series_operations
    Comprehensive time series analysis including trend detection, seasonality analysis, and forecasting
    Parameters: file_path (str), date_column (str), value_columns (list), operations (list), frequency (str, optional)
  • validate_data
    Comprehensive data validation with quality checks, constraint validation, and data integrity assessment
    Parameters: file_path (str), validation_rules (dict), output_file (str, optional)
  • filter_data
    Advanced data filtering with sophisticated boolean indexing and conditional expressions
    Parameters: file_path (str), filter_conditions (dict), output_file (str, optional)
  • optimize_memory
    Optimize memory usage with data type optimization and efficient storage strategies
    Parameters: file_path (str), optimization_level (str, optional), output_file (str, optional)

    Generate comprehensive data profiling reports with quality assessment and insights
    Parameters: file_path (str), columns (list, optional), include_correlations (bool, optional)

Usage Examples

Basic Data Analysis

result = load_data("/data/employees.csv", encoding="utf-8") stats = statistical_summary("/data/employees.csv", columns=["salary", "age", "years_experience"], include_distributions=True)

Data Cleaning Pipeline

missing_handled = handle_missing_data("/data/dataset.csv", strategy="mean", columns=["price", "quantity"]) cleaned = clean_data("/data/dataset.csv", remove_duplicates=True, detect_outliers=True, convert_types=True)

Advanced Transformations

grouped = groupby_operations("/data/sales.csv", group_by=["region", "category"], operations={"revenue": "sum", "quantity": "mean"}) pivot = pivot_table("/data/sales.csv", index=["date"], columns=["category"], values=["revenue"], aggfunc="sum")

Time Series Analysis

resampled = time_series_operations("/data/timeseries.csv", date_column="timestamp", operation="resample", frequency="M") rolling = time_series_operations("/data/timeseries.csv", date_column="timestamp", operation="rolling_mean")
