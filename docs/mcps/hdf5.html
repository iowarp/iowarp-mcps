HDF5

Data Processing

List and analyze HDF5 files for hierarchical data structures. Provides efficient access to large scientific datasets stored in HDF5 format.

1.0.0 Version
4 Actions
Jan 15 Updated
Python Language

Installation Methods

Global Installation

uv pip install scientific-mcps

Individual MCP Installation

uv pip install "scientific-mcps[hdf5]"

Direct Server

uv run mcp_hdf5

Claude Desktop Configuration

{ "mcpServers": { "hdf5": { "command": "uv", "args": ["run", "mcp_hdf5"] } } }

Available Actions

  • list_hdf5
    List HDF5 files in a directory
    Parameters: directory (str, default="data/")
  • inspect_hdf5
    Inspect HDF5 file structure: lists groups, datasets, and attributes
    Parameters: filename (str)
  • preview_hdf5
    Preview first N elements of each dataset in an HDF5 file
    Parameters: filename (str), count (int, default=10)
  • read_all_hdf5
    Read every element of every dataset in an HDF5 file
    Parameters: filename (str)

Usage Examples

HDF5 File Analysis

// List HDF5 files list_hdf5(directory: "/data/experiments/") // Inspect file structure inspect_hdf5(filename: "/data/experiment_data.h5") // Preview first 10 elements of each dataset preview_hdf5(filename: "/data/experiment_data.h5", count: 10) // Read all data from file read_all_hdf5(filename: "/data/small_dataset.h5")

Data Processing Pipeline

// Explore HDF5 structure inspect_hdf5("/data/simulation_output.h5") // Preview datasets for quick analysis preview_hdf5("/data/simulation_output.h5", 5) // Read complete dataset for processing read_all_hdf5("/data/processed_results.h5")
